# Line Follower Robot

Autonomous line follower robot developed using Arduino Nano, IR sensor array, TB6612FNG motor driver, and PWM-based speed control for real-time path tracking.

## Features
- Real-time line detection using 5-channel IR sensor array
- Differential steering using dual DC motors
- PWM motor speed control
- Stable regulated power using buck converter
- Expandable sensor integration (color sensor)

## Components Used
- Arduino Nano
- TB6612FNG Motor Driver
- IR Sensor Array
- N20 DC Gear Motors
- Li-ion Battery
- Buck Converter
- Chassis and Wheels
